# Visuelum — Next.js 14 Website

Premium digital marketing agency website built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS v3
- **Animations:** Framer Motion
- **Forms:** React Hook Form + Zod
- **Email:** Resend
- **Deployment:** Vercel

## Getting Started

```bash
pnpm install
pnpm dev
```
